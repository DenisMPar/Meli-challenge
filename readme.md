# Meli - Clon

Clon de la página de detalle de producto de **Mercado Libre** la cual muestra detalle del producto, descripción, opiniones y puntuaciones de los usuarios.

Debido a que la API de mercado libre dejó de ser pública, este proyecto solo muestra datos “Mockeados”

## **Tecnologías Usadas:**

- Vite.
- React.
- Typescript.
- Chakra-Ui
- Recoil
